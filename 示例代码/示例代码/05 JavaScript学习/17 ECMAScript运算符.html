一元运算符

  • delete

    delete 运算符删除对以前定义的对象属性或方法的引用。 delete 运算符不能删除开发者未定义的属性和方法,否则将引发错误。

  • void

    void 运算符对任何值返回 undefined。 该运算符通常用于避免输出不应该输出的值，例如，从 HTML 的a 元素调用 JavaScript 函数时。要正确做到这一点，函数不能返回有效值，否则浏览器将清空页面，只显示函数的结果。 提示：请记住，没有返回值的函数真正返回的都是 undefined。
    未使用void，点击后原页面显示[object]
    使用void，点击后原页面不变;
